# curso_git

Repositório do curso de git na Udemy

## Repo de testes e estudo apenas

### Comandos básicos do git
* git commit
* git push
* git pull
* git clone
* git merge
* git checkout

### Comandos de rotina
* git merge main // somente usar em uma branch
* git pull
* git gc
* git fsck

### Comando tag
* git tag
* git tag -a v1.0 -m "comentário"
* git show {tagName}
* git checkout {tagName}
* git push origin {tagName}
* git push origin --tags

### Comando stash
* git stash
* git stash list
* git stash apply {id}
* git stash show -p {id}
* git stash drop {id}
* git stash clear

### Comando log
* git log
* git short log
* git reflog

### Outros comandos
* git show
* git diff
* git clean
* git fetch -a
* git archive --format zip --output arquivos_main.zip main
* git rebase
